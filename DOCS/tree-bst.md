## 二分搜索树（Binary Search Tree）

### 大O复杂度

| 操作     | 时间复杂度 | 空间复杂度 |
| -------- | ---------- | ---------- |
| 插入元素 | O(LogN)    | O(1)       |
| 修改元素 | O(LogN)    | O(1)       |
| 删除元素 | O(LogN)    | O(1)       |
| 查找元素 | O(LogN)    | O(1)       |
